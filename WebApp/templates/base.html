Parallelisation of UCR Dynamic Time Warping Implementation

The aim of this application is to compare the performances of the UCR implementation of the Dynamic Time Warping (DTW) algorithm with a multi core implementation.
The application allows you to select a dataset of a particular size and a query of randomly generated points to be used in both implementations.
The output is a graphical respresentation of the matched sequence of the two time series along with location, distance and time consumed by each of the implementations.
The available datasets are as follows:

Input Options:

  • 500MB - 40,000,000 points
  • 1GB - 80,000,000 points
  • 3GB - 2,20,000,000 points
  • 6GB - 4,40,000,000 points
